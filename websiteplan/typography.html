Heading Font: Ubuntu

I chose Ubuntu as a heading because it's a little more smooth but still catches your attention. It sticks out, but not too much.

Paragraph Font: Nunito

I chose this font for the paragraphs (and just about everything else) because it's smooth but it isn't quite as smooth as the heading font. It's also a little thiner by default which helps create a hierarchy.
